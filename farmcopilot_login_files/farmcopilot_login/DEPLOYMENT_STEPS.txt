1) Open js/supabase-config.js
2) Replace REPLACE_WITH_FULL_SUPABASE_PUBLISHABLE_KEY with your full Supabase publishable key
3) Upload auth.html and the full js folder to your project
4) Add this script at the top of protected pages like dashboard.html:
   <script src="js/auth-guard.js"></script>
5) In Supabase > Authentication > URL Configuration, add:
   Site URL: https://your-domain.com
   Redirect URLs:
   https://your-domain.com/auth.html
   https://your-domain.com/dashboard.html
6) If using Google sign-in, enable Google provider in Supabase Authentication > Providers
7) Do not place service_role keys in any frontend file
